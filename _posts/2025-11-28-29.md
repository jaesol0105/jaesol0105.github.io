# 2025-11-28

### API 키는 기본적으로 Github에 올리면 안된다.

## dotenv
- API 키 관리를 위한 패키지
- 프로젝트내에 env파일을 만들고, API 키 등의 Github에 올라가면 안되는 민감한 정보들을 다룬다
- env파일은 git ignore 해야하고 직접적으로 주고 받아야한다
